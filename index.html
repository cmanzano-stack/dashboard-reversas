Configuración de conexión
Modo
Ollama corre en tu máquina en localhost:11434. Para permitir que el dashboard lo use desde el navegador, inicia Ollama con:

OLLAMA_ORIGINS=* ollama serve

En Windows usa PowerShell:
$env:OLLAMA_ORIGINS="*"; ollama serve
Ingresa la URL de tu servidor Ollama. Asegúrate de que el servidor tenga CORS habilitado:
OLLAMA_ORIGINS=* ollama serve
Configura y prueba la conexión con Ollama, luego haz una consulta para analizar los datos.
